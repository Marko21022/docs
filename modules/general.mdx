## General module
These are global settings that are not categorized
<Frame caption="General module settings"><img src="/images/modules/general.png" style={{height: "auto", width: "750px", borderRadius: "10px"}}/></Frame>
<b>Options for general module:</b>
- <Icon icon="file-text" /> <span style={{ color: "white" }}>System Pessages</span>
<Note>Configuring this option is required</Note>
- <Icon icon="pencil" /> <span style={{ color: "white" }}>Custom prefix</span><br />
Use this option to configure the prefix for message commands. The default prefix is `>`, and by leaving that option empty the bot will use the default prefix
## Custom Profile
<Callout icon="star" color="#FFC107">
This feature requires <b>Sway Premium</b>.
</Callout>
<Frame caption="Custom profile settings"><img src="/images/modules/profile.png" style={{height: "auto", width: "750px", borderRadius: "10px"}}/></Frame>

- <Icon icon="text-cursor-input" /> <span style={{ color: "white" }}>Name</span><br />
Set the display name for Sway in the selected server
- <Icon icon="square-user" /> <span style={{ color: "white" }}>About me</span><br />
Customize the about me section of Sway bot in the selected server
- <Icon icon="image" /> <span style={{ color: "white" }}>Profile Picture</span><br />
Upload a custom profile picture that Sway bot will use in the selected server
- <Icon icon="image" /> <span style={{ color: "white" }}>Profile Banner</span><br />
Upload a custom banner that Sway bot will use in the selected server<br />

<Danger>By clicking <b>Reset Profile</b> button in the top right corner, you will be able to reset the bot profile to the default one</Danger>
<Note>Updating the bot's profile may take a while</Note>
